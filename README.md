## 📌 Overview

**Solar System** is a professional engineering tool designed to
accurately size and design solar power systems for residential and small
commercial use.

It analyzes real-world energy consumption and intelligently generates
optimal configurations for: - Solar panels ☀️\
- Inverters 🔌\
- Batteries 🔋

The goal is to deliver **efficient, reliable, and cost-effective solar
solutions**.

------------------------------------------------------------------------

## ✨ Features

### ⚡ Intelligent System Design

-   Automatic solar system sizing
-   Based on real consumption (Wh/day)

### 🔍 Smart Component Selection

-   Matches best components automatically
-   Database includes:
    -   17+ inverters
    -   18+ batteries

### 📊 Engineering-Level Accuracy

-   Uses:
    -   Peak Sun Hours (PSH)
    -   System loss factor
-   Ensures realistic results

### 🧠 Advanced Filtering Logic

-   Inverter selection based on:
    -   AC Power (kW)
    -   Max PV Input (W)
